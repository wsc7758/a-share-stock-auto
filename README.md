# A股题材热点自动分析系统
> ⚠️ 极度重要免责
本项目仅作为Python量化编程学习工具，所有自动生成的资讯、指标、选股名单、支撑压力、交易策略**不构成任何投资建议！**
禁止用于荐股、收费分享，股市风险自负。

## 功能链路
1. 抓取财联社快讯 + 板块资金，量化计算热点板块
2. 全局市场情绪打分动态修正板块评分
3. 板块内多因子选出龙一/龙二/龙三
4. 自动识别标的护城河属性
5. 移植InStock算法：N字形态、箱体突破、筹码测算
6. 技术指标计算支撑压力、自定义买卖模型
7. 根据情绪、龙头顺位、护城河动态生成交易策略
8. GitHub Actions 交易日自动运行，归档每日结果

## 运行方式
GitHub Actions 定时自动执行，也可本地运行 `python task/run_daily.py`
